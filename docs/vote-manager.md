# Vote Manager integration

Сообщение владельцу сервера:

> Привет! Для автоматической выдачи бонуса за голос на L2Realm настройте у себя Vote Manager.
>
> Игрок голосует на странице вашего проекта на L2Realm и вводит ник персонажа. После этого ваш Vote Manager должен проверить голос запросом:
>
> `GET https://l2realm.ru/api/vote/check?server_id=SERVER_ID&nickname=PlayerNick`
>
> Где `SERVER_ID` — id проекта в L2Realm, `PlayerNick` — ник игрока на вашем сервере.
>
> Пример ответа, если голос найден за последние 24 часа:
>
> ```json
> { "voted": true, "timestamp": "2026-05-09T14:00:00.000Z" }
> ```
>
> Если голоса нет:
>
> ```json
> { "voted": false, "timestamp": null }
> ```
>
> Если `voted: true`, выдавайте бонус игроку на своей стороне. Чтобы один голос не забирали несколько раз, храните у себя `nickname + timestamp` или используйте стандартную защиту Vote Manager от повторной выдачи.

Notes:
- L2Realm ограничивает голосование: 1 голос с одного IP за 24 часа и 1 голос с одного авторизованного аккаунта за 24 часа.
- Endpoint публичный и предназначен только для проверки голосов, он не выдаёт бонус сам.
