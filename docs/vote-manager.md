# Vote Manager Integration

Короткий текст, который можно отправлять владельцу сервера после добавления проекта на L2Realm.

## Сообщение Владельцу

Привет! Ваш сервер добавлен в каталог L2Realm.

Если хотите автоматически выдавать бонусы игрокам за голосование, подключите у себя Vote Manager к нашему API.

Игрок голосует на странице вашего проекта на L2Realm и вводит ник персонажа. После этого ваш Vote Manager проверяет голос запросом:

```http
GET https://l2realm.ru/api/vote/check?server_id=SERVER_ID&nickname=PlayerNick
```

Где:
- `SERVER_ID` - id вашего проекта в L2Realm.
- `PlayerNick` - ник игрока на вашем сервере.

Если голос найден за последние 24 часа:

```json
{ "voted": true, "timestamp": "2026-05-09T14:00:00.000Z" }
```

Если голоса нет:

```json
{ "voted": false, "timestamp": null }
```

Если `voted: true`, выдавайте бонус игроку на своей стороне. Чтобы один голос не забирали несколько раз, храните у себя пару `nickname + timestamp` или используйте стандартную защиту Vote Manager от повторной выдачи.

## Важные Правила

- L2Realm ограничивает голосование: 1 голос с одного IP за 24 часа.
- Для авторизованных пользователей также действует ограничение: 1 голос с аккаунта за 24 часа.
- API только проверяет голос, но не выдает бонус сам.
- В админке проекта есть галочка `Vote Manager / Бонусы за голос подключены`.

## Где Взять SERVER_ID

`SERVER_ID` виден в URL/API проекта. Если владелец не понимает, какой id использовать, проще отправить ему готовую ссылку проверки уже с id его проекта:

```http
https://l2realm.ru/api/vote/check?server_id=SERVER_ID&nickname=PlayerNick
```
